Esta API permite encriptar y desencriptar datos utilizando el algoritmo **AES** en modo **CBC**.

## Requisitos

- **Python 3.7+**
- **FastAPI**
- **Uvicorn**
- **cryptography**

Instala las dependencias con:

```bash
pip install fastapi uvicorn cryptography
```

## Levanta el servidor:
```bash
uvicorn main:app --reload
```
